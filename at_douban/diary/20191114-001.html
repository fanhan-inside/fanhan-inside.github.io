［20191114］又到了劳逸结合的时候了

刚才花了一个小时投入到业余爱好之上，看完了《微分流形初步》的第一章〈预备知识〉，回顾了「张量」「外代数」等内容，按照惯例又特么的开始头疼了，所以赶紧从事政治和意识形态斗争的本职工作，调节一下身心健康。

结合最近党国高音喇叭的高调报道和民间色目情报掮客的不打自招，重复的内容不提，先问问各位读者的感想：看到包括但不限于三洲通衢和内亚多地，动辄出现自杀式恐怖袭击还尚未有组织或个人宣布负责，经常死了十几个伤了几十个……这是「鼠标战神」在玩游戏么？

如果认为不是，那就不要用娱乐至死の话语体系解读，比方说在《知乎问答№084》当中援引色目素材所记载的「国家不幸玩家幸」的二十年目睹之怪现状。

肯定有根红苗正忠君爱国的童年才俊不服，说什么本职工作是高举紧跟深入学习系列重要讲话精神并写出篇幅充沛的心得体会，为了调解身心健康才选择人民群众喜闻乐见的休闲方式，五大流氓自有实力在此，崛起于世界民族之林乃不可阻挡の兲命气数，跳梁小丑不自量力螳臂挡车钦定被历史の五对负重轮碾为齑粉……什么的。

也就是说，境内河清海晏歌舞升平，没出现啥无人机投掷石墨炸弹破坏电网之类恐怖袭击，顶多在那兲子脚下首善之区夜不闭户路不拾遗的国际一流和谐宜居之都，孤立的突发的偶然的片面的存在色目混混鸡鸣狗盗之徒の涨落现象，并且随机的选择人民群众之住址，掐断其网线并关闭其电源……而已。

这话倒也没说错，所谓「王不见王」就是大流氓在台面上各个和颜悦色谈笑风生，暗地里进行「代理人战争」，专挑穷乡僻壤犄角旮旯作为「场地」，并在「裁判」的监督下「友谊第一比赛第二」，所以才会出现那许多「NPC」作为各路「勇者」团队的「目标」至少是「战利品」。

好了，接下来就是色目情报掮客所暗示的剧透了，既然是代理人战争，那么肯定有目前现居国际一流和谐宜居之都指挥色目混混鸡鸣狗盗之徒绕着俺様团团转从事「超限战」的高第良将「鼠标战神」之代理人，在那「一带一路」和广袤的「撒哈拉以南非洲」地区，专挑兲朝援建的「大国工程」，以无人机投掷石墨炸弹，进行「无限博弈」直到兲朝耗尽「资源和意志」，一个响头磕到白宫面前宣布「贸易战」无条件投降。

理由都说过了，兲朝那充沛的武德只能窝里横，没有全球投送能力，对通过国际一流和谐宜居之都色目混混鸡鸣狗盗之徒经过实践所检验的成功案例照搬过去「性价比」极高的「超限战」毫无还手能力，摊子铺的越大麻烦就越多，会在换届前的三年间被拖入「千日防贼」的军备竞赛，还是左支右绌单方面挨打的窘境。

再看耍钱的朋友们奔走相告的重要情报，尤其是情报工学神童贵种大少爷所重点关注的，听说最近阿里集团要回归港股上市，与腾讯唱对台戏，以确定谁是刚刚被南不列颠及北爱尔兰交还圆明园的「马首」。

按照退而不休的马云讲话，阿里早就不是「电商」企业了。在充沛着国家机密兼宇宙奥秘的境内，难以获得精确的捅妓情报，这话暂且按下不表，可以找个类型相同至少是相似的替代样品进行分析，比方说美帝灯塔国的亚马逊。

最新的关于亚马逊的情报，是负面消息，说美帝灯塔国の武德在一百亿美元的云计算招标当中选择了微软而不是高调叫嚣全面转型升级「不再是电商企业」的亚马逊。业内对此有很多解读，既有技术上的，也有经济上的。

从技术上说，业内爆料这些曾经的「电商」企业，即便在访问高峰期间，比如刚过去的光棍节促销，或者即将到来的耶少爷诞辰纪念日促销，也有至少一半的计算资源「闲置」。据称理由是业务为带宽密集型，以数据库操作为主，网卡超负荷运转，但是处理器则不是，并且这个矛盾还不可调和。

于是叫嚣云计算最凶的，除了亚马逊就是阿里了，一块抹布一张卫生纸都有它的价值，非得把闲置资源推销出去不可。这种时候，美帝灯塔国の武德对于亚马逊的成功经验表示怀疑，认为其在「计算」上的技术储备还不如微软，所以才作出了未必民主但是自由の抉择。

从经济上说，色目情报掮客曾经炫耀过特大日本帝国「兵库商社」的成功经验，说在关西机场建设过程中需要大量土石方，也是面向全国招标，唯独兵库县的报价比其它竞争对手低了一半以上，据称是同时正在开展大县工程到处大兴土木，土石方极大充沛于是变废为宝……什么的。

色目鸡汤推销者还说，如果「兵库商社」的报价仅仅比其它竞争对手低一点，好比境内手眼通天的豪门贵种走兽派，经常可以伸出一根小指头点一下鼠标即可获得所有其它散户韭菜的商业机密，开了「透视挂」战无不胜攻无不克，估计建设方会认为「没有诚意」，品质也不如其它专门开采土石方的投标者，除非一锤子买卖只要「比赛」胜利就「赢者通吃」，否则在「无限博弈」当中还是略微讲一点「职业道德」比较好。

同理可证，估计这次投标当中亚马逊的报价，与微软拉开的差距并不大，于是被美帝灯塔国の武德判断为「没有诚意」或者「发国难财」之类，不愿意帮忙解决其「库存」之后，还被色目媒体工作者嘲笑为「自带干粮卖命」「凭本事骗来的钱凭什么要还」「被咱哄了还替咱数钱」，这才稳妥起见，选择了「专门」搭建云计算平台而不是以边角料凑数的微软。

纯粹学术角度的解读不提了，比方说阿里和亚马逊都以Java™技术栈为基础，微软则提供.Net Core什么的。去年两会结束之后，在国际一流和谐宜居之都「高学历精英社交圈」谈笑风生了两篇《知乎问答№151：.NET 开源对 Java 的冲击有多大？》《知乎问答№152：PostgreSQL 与 MySQL 相比，优势何在？》，立刻就被站务以「政治敏感」理由「亦当删去」。

当时我还质问过，纯粹学术角度的回答为啥会「政治敏感」呢？而知乎首页今日头条位置醒目应景推荐的一连串「震惊」「重磅」内容当中，就出现了「智多星」高瞻远瞩算无遗策于是人缘差情商低的谁被识大体顾大局懂政治讲规矩守纪律的党国栋梁青年才俊「玩死了」以及兲朝自有国情在此「鸡巴尊贵就是可以为所欲为」相关内容。

现在各位参考美帝灯塔国の武德的选择，知道技术栈选型方面的问题，在境内到底「敏感」在哪里了么？如果还有人缘好情商高识大体顾大局懂政治讲规矩守纪律的童年才俊惦记着在情报工学领域与神童贵种大少爷「竞争」，可以参考这个商业案例。

结合南直隶「远东第一体育论坛」涌现的色目文艺爱好者的爆料，以查博导知名话本当中华山派内乱影射「气宗」「剑宗」之间的内讧，已经解读了信息技术领域的「学术纠纷」转变为「路线斗争」的必然性，并且还文斗不如武斗，成王败寇，胜利者不受谴责，各种阴谋诡计层出不穷，只有在「历史由胜利者书写」的时候才大事化小小事化了的洗白为「校园欺凌」什么的。
