微软 Microsoft Build 2018 上公布了哪些不负众望的产品、技术或合作？

美东时间5月7日8:30(北京时间23:30)，为期三天的微软开发者大会Build 2018在美国华盛顿州西雅图市拉开帷幕。Microsoft Build是微软面向全球开发者、合作伙伴以及消费者介绍其重要产品在未来一年内发展方向的技术盛会。

范狂夫

来，入乡随俗，抖个机灵。

刚才知乎人工智障算法于首页今日头条位置应景推荐的是这篇文章：

Build 2018 Day 2 剧透+总结：基于 Microsoft 365 跨平台 Windows 体验

然后里面有个关键字「.Net Core 3.0」，选中后右键菜单执行「搜索－Bing」。

用微软自家搜索引擎查自家产品，返回结果总不至于出现一堆推销楼盘、高利贷、「投资理财」、「保健品」、网游……的广告了吧？

第二条结果就是微软官方站点博客：

.NET Core 3 and Support for Windows Desktop Applications

里面有这么一段话：

We have a version of Paint.NET running in our lab. In fact, we didn’t have access to Paint.NET source code. We got the existing Paint.NET binaries working on .NET Core. We didn’t have a special build of WPF available, so we just used the WPF binaries in the .NET Framework directory on our lab machine. As an aside, this exercise uncovered an otherwise unknown bug in threading in .NET Core, which was fixed for .NET Core 2.1. Nice work, Paint.NET!

最后一句是我加粗的，信达雅的翻译过来就是「厉害了，.Net Core！」「为Paint.Net点赞！」

召唤党国栋梁童年才俊组织@共青团中央向中宣部邀功请赏，连亡我之心不死的帝国主义白皮鬼子都与时俱进拿来主义了吖，可见革命口号循环播放之刷屏效果好于预期。

但是不能骄傲自满，还得再接再厉，赶紧划拨预算「大力」开发人工智障大数据应用，争取把新时代中国特色社会主义大好形势下繁荣的中文互联网上所有页面广告都替换成这两句革命口号。

……球安民乐岂不美哉？！
