如何评价Nano在2016年6月17日离开GNU，9月1日重回GNU？

Nano is no longer a GNU project

Nano is again a GNU project

2016 June 17

nano 2.6.0 “Rubicon” fixes more than fifty little bugs -- and some of them not so little. It improves moving about in the file browser, corrects failings of the internal spell checker, adds a new feature (comment/uncomment lines, with default binding M-3), makes some error messages clearer, shows more of a file when positionlog is used and the cursor is near the end, displays all error messages at startup if there are multiple ones, does not misinterpret keystrokes when typing very fast, is less eager to trim the filename on narrow terminals, speeds up case-insensitive searches, and allows to abort re-searches. Among bunches of other things. It is worth the trouble to upgrade.

And, with this release, we take leave of the herd...

Bye! And thanks for all the grass!

2016 September 1

GNU nano 2.7.0 “Suni” adds a new feature: allowing text to be selected by holding Shift together with the cursor keys. Besides that, nano now works also when run in very tiny terminals (down to one line, one column), and improves the handling of the prompt in cramped spaces. Not much, but it’s time to get it out there.

With this release we return to GNU. For just a little while we dreamt we were tigers. But we are back in the herd, back to a healthy diet of fresh green free grass.

范狂夫

这其中一定蕴涵着充沛的政治和意识形态内容并体现了政治和意识形态斗争新动向。

使用「一定」字眼进行不惮以最大的恶意揣测活动，是因为已经过去了两年多，当时没有注意到的「业界动态」也难以追溯，不过「时事政治」汇总倒是可以在网上查到。

问题写出的「6月17日」是版本发布日期，而给出的链接当中「脱离GNU」的公告则是「6月22日」，这两个日期其实都有重要意义。

回顾2016大事记：

英国主张留在欧洲联盟的女议员乔·考克斯6月16日在英格兰北部西约克郡参加选区活动时遭枪击身亡，成为1990年以来首名被杀害的在任英国议员。

英国当地时间6月23日就脱离欧盟举行全民公投。24日上午综合全部计票结果显示，“脱欧”一方支持率为51.89%，而赞成“留欧”的投票者占48.11%。

于是，这个项目就在那「6月17日」发布了「新」版本，而在脱欧公投前夕发布「脱Gnu」公告。

至于为啥在「9月1日」宣布回归，虽然难以找到确切的标志性新闻，但是想来也应该与「南不列颠及北爱尔兰联合王国」相关动向有关。

顺便，在常见简单任务当中被这个Nano所替代的Vim，其启动画面中会显示作者号召给乌干达饿殍捐款的信息，想必是倾向于支持尼格罗太君或者曾经获得相关阵营赞助。

因此，Nano的脱离会让Gnu当中Vim所占比例更大，是否暗示着「难民迁移改变人口结构」，与「尼格罗太君团结振兴的非洲梦」有关呢？

最后，还是强调之前说过多少次的关键，在知乎首页出现的各种「真理标准大讨论」，哪怕是信息技术相关话题，都蕴涵着知乎管理层所关注的充沛政治和意识形态内容并体现了知乎管理层在兲朝党政军财各个领域进行政治和意识形态斗争的新动向。

就说编辑器，Vim作者倾向尼格罗太君，而Emacs作者则是赛博朋克英特纳雄耐尔主义者Gnu元勋，这个Nano也是一样。于是相应的用户群在「真理标准大讨论」若是使用超出技术角度描述的措辞，尤其是上纲上线的时候，就会被各种「捅妓机关」（包括但不限于前朝「中统」「军统」本朝「国统」）拿来作为「民意调查」的「大数据」使用。
